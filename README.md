# duckie!

Hello! Welcome to my first PCB, powered by Solder!

My slack username: @shreyaluu

This is just a little duck keychain whose eyes light up when the button is pressed.

### Bill of Materials
- 1x PCB
- 1x 3V Battery
- 2x LEDs
- 2x 220 Ohm Resistor
- 1x Battery Holder
